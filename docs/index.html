Exploring Radar Basics: A Hands-On Guide to the EVALKIT SiRad Simple®

Francesco Mancuso published on 2024-01-15
Exploring Radar Basics: A Hands-On Guide to the EVALKIT SiRad Simple® The EVALKIT SiRad Simple® is an evaluation board from Silicon Radar (which has now been acquired by Indie Semiconductor actually) designed to use and test one of their FMCW Radar-on-Chip sensors. The evaluation kit communicates via UART and features a ARM Cortex-M4 microcontroller. Though this evaluation board is no longer on the market, there’s still good reason to talk about it.
